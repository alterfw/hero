Hero

Hero is the Alter's main module, responsible for all the interaction with post types and taxonomies throught models.

Instalation

composer require alterfw/hero

Usage

Using Hero out of the box.

<?php

$hero = new Hero();
$app = $hero->get();

$books = $app->books->find();
